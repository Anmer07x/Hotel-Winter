# Hotel Winter 🏨

**Hotel Winter** es una aplicación web desarrollada como proyecto universitario para la gestión completa de un hotel. Fue construida con el framework **Laravel** y permite realizar reservas, gestionar habitaciones, usuarios y más.

## ✨ Funcionalidades principales

- Registro y login de usuarios.
- Gestión de habitaciones (crear, editar, eliminar).
- Sistema de reservas.
- Panel administrativo con control de usuarios.
- Diseño intuitivo y responsivo.
- Control de disponibilidad y fechas.

## 🛠️ Tecnologías utilizadas

- Laravel 10
- PHP 8
- MySQL
- Blade (motor de plantillas)
- Bootstrap
- JavaScript

## 📚 Objetivo del proyecto

Este sistema fue desarrollado como parte de un proyecto académico universitario con el objetivo de aplicar conocimientos prácticos en desarrollo backend con Laravel y estructuración de bases de datos relacionales.
